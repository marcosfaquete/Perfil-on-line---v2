# 🚀 MF/DEV - Professional Portfolio

Este é o meu portfólio pessoal de alta performance, desenvolvido para apresentar minha trajetória como desenvolvedor e especialista em infraestrutura de TI. O projeto utiliza as tecnologias mais modernas do ecossistema React para entregar uma experiência de usuário (UX) premium, fluida e focada em conversão.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído com uma stack moderna focada em escalabilidade e performance:

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/) (Tipagem estrita para maior segurança)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/) (Design responsivo e utilitário)
- **Backend & Database:** [Supabase](https://supabase.com/) (Real-time analytics e persistência de dados)
- **Componentes & UI:** [Lucide React](https://lucide.dev/) & [Framer Motion](https://www.framer.com/motion/) (Animações e ícones)
- **Deployment:** [Vercel](https://vercel.com/)

---

## ✨ Funcionalidades Principais

- **Bento Grid Design:** Interface inspirada em dashboards modernos, organizando informações de forma hierárquica e visual.
- **Real-time Counter:** Integração com Supabase para exibição de visualizações em tempo real.
- **Glassmorphism UI:** Efeitos de transparência e desfoque de fundo (backdrop-blur) para uma estética "SaaS Premium".
- **Micro-interações:** Animações contínuas e efeitos de hover neon (Glow) nos cards e tech stacks.
- **Foco em Conversão:** Call-to-Actions (CTAs) estratégicos integrados diretamente ao WhatsApp.
